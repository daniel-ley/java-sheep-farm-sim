# Sheep Farm Simulator

A cosy farm simulator game to practice Java.

## Current Features
- Sheep class with basic attributes (name, colour, age, satiety)
- Flock management using ArrayList
- Basic behaviours: eating, growing, bleating

## Planned Features
- Border collie herding mechanics
- Pasture management
- Wool production and shearing
- A farm cottage to decorate
- Wool dying and yarn crafts

## Running
Requires Java 21 or higher.
